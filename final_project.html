It's Written in the Stars:

By Ashley Yu and Dana Li

As part of Yelp’s Dataset Challenge, they have made the data for over 77K businesses available. Attributes of the API include categories, location, price range, stars, etc. We have taken the raw data and parsed it down to one city (Las Vegas) and one type of business (restaurants) via server side pre-formatting, resulting in approximately 5K entries.

We have created an interactive data visualization of the restaurants in Las Vegas, which allows users to explore the data for themselves in an aesthetically pleasing way. Whether they wish to quickly determine viable restaurants to visit or draw conclusions about trends, they are able to do so through dynamic filtering on attributes and view relevant charts with the number of reviews, price range, and information about the businesses. The charts are built with DC.js and Crossfilter, and they allow for a quick view of trends and drill downs on specific attributes. The word cloud is a customized D3 visualization that takes the categories attribute from each data point and displays them, with size corresponding to the number of occurrences in the set. The word cloud re-calculates its visualization each time a unique filter is applied. The map visualization shows the density of businesses in various districts and trends by location. The map also dynamically updates as filters are applied, and can itself serve as a filter based on zoom views. Finally, a data table shows a list of restaurants and their relevant information, depending on what’s been filtered.

Libraries used include:
DC.js (Version: 1.7.5), D3.js (Version 3.5.6), Underscore.js (Version: 1.8.3), and Crossfilter.js (Version: 1.3.12)
